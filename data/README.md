# Data Directory

## Dataset Source

Download the Black Friday Sales dataset from Kaggle:
**https://www.kaggle.com/datasets/sdolezel/black-friday**

## Setup Instructions

1. Download the dataset from Kaggle (requires Kaggle account)
2. Extract the files
3. Place the CSV files in the appropriate directories:

```
data/
├── raw/
│   ├── train.csv          # Download from Kaggle
│   └── test.csv           # Download from Kaggle
├── processed/             # Generated by notebooks (Delta Lake format)
└── streaming/             # Generated by streaming pipeline
```

## Dataset Information

**Size:**
- train.csv: ~25 MB (~550,000 rows)
- test.csv: ~9 MB

**Schema:**
- User_ID
- Product_ID
- Gender
- Age
- Occupation
- City_Category
- Stay_In_Current_City_Years
- Marital_Status
- Product_Category_1
- Product_Category_2
- Product_Category_3
- Purchase (target variable)

**Note:** Raw CSV files are excluded from git (.gitignore) due to size.
